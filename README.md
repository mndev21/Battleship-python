# Terminal Battleship Game

## Project Overview

This is a simplified terminal version of the classic Battleship game. The game features:

- A 10x10 board for both the player and the bot.
- Ship configuration:
  - 1 ship of size 4
  - 2 ships of size 3
  - 3 ships of size 2
  - 4 ships of size 1
- Ships cannot touch each other, even diagonally.
- Both the player and the bot place their ships according to the same rules.
